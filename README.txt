This is a Plugin for Wordpress using Woocommerce Xero Extension and Stripe Gateway. Converts your order into the currency that Stripe is using for Xero invoices. Requires the 'Woocommerce Xero Stripe Fees' plugin (https://github.com/dfinnema/woocommerce-xero-stripe-fees)

== Description ==

This is a Plugin for Wordpress using **Woocommerce** with the **Woocommerce Xero** Extension and **Stripe**

It converts your order into the currency that Stripe is using for Xero invoices

**Please note this is a work in progress and probably has a lot of bugs in it. The code is quick and dirty so feel free to add / change to it.**

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `woocommerce-xero-stripe-currency` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change settings under WooCommerce > Xero
